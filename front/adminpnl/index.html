 Biometrics

Biauthorize achieves two factor security without a physical token. In the past, remote biometric authentication has been tedious and complex. Biauthorize changes that, with an easy to use API and a functional authentication structure.

 Secure

Biauthorize effectively ensures that the user is who they claim to be, not an imposter. With Biauthorize, a user can't share their two factor token with someone else. Unless the user is physically at the computer system, they will not be able to log in.

 Easy

Biauthorize allows you to quickly and seamlessly bring biometric authentication to your web application. By adding a single line of JavaScript to your login form, you can give your users the added security of Biauthorize.

Can I use Biauthorize for three factor authentication?

Yes, Biauthorize can be used in conjunction with other two factor authentication systems.

Is Biauthorize secure?

It's almost impossible to completely secure a networked application. Biauthorize checks multiple images during the same session to minimize the risk of basic attacks like holding up a photo of the victim. Biauthorize also checks to make sure an image is not submitted more than once. Biauthorize is particularly effective for preventing the use of captured login credentials, such as via SSL stripping. Even if an attacker captures a user's password, it is highly unlikely that they will be able to impersonate that user's face.

Do you have code examples?

Yes, verification examples are available in PHP, Node.js, Python, and Ruby.
